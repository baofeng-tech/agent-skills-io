# X Intelligence Automation

Cross-platform Twitter/X research, posting, and engagement skill for AgentSkills-compatible clients.

## What it does

- Search Twitter/X content and monitor accounts or topics
- Publish through OAuth after explicit approval
- Perform approved engagement actions like like, unlike, follow, and unfollow

## Setup

```bash
export AISA_API_KEY="your-key"
```

Requires:

- `python3`
- network access to `https://api.aisa.one/apis/v1/...`
